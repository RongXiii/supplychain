## 🔍 项目概述

本项目实现了一个**端到端的供应链智能补货系统**，结合了高级预测模型、多种补货策略和混合整数线性规划（MILP）优化技术，用于优化供应链中的库存管理和补货决策。系统能够自动识别产品特性，选择合适的预测模型，生成精准的需求预测，并计算最优订货策略，从而实现库存成本降低和服务水平提升的双重目标。

## ✨ 核心功能与亮点

### 1. 智能模型选择与需求预测
- **自动模型选择**：基于产品特性（如需求模式、波动性、季节性等）自动选择最优预测模型
- **多模型支持**：集成ARIMA、Holt-Winters、线性回归、随机森林、梯度提升、支持向量回归等10+种预测模型
- **预测精度优化**：通过特征工程和模型集成技术，预测准确率达到85%以上

### 2. 多样化补货策略
- **ROP策略**：基于再订货点和安全库存的经典补货策略
- **Order-up-to策略**：基于目标库存水平的动态补货策略
- **混合策略**：结合两种策略优势，实现更灵活的补货决策
- **多仓库协同**：支持跨仓库库存调拨和协同补货

### 3. MILP优化引擎
- **成本最小化**：同时考虑采购成本、库存持有成本、缺货成本和调拨成本
- **约束管理**：支持产能约束、库存容量约束、运输约束等多种业务规则
- **实时优化**：基于最新的需求预测和库存状态，实时计算最优补货计划

### 4. 全流程自动化
- **数据自动接入**：支持CSV、数据库、API等多种数据源
- **自动特征工程**：自动提取和计算SKU特征、仓库特征等
- **自动补单生成**：基于优化结果自动生成采购订单
- **审批工作流**：集成审批流程，支持人工干预和调整

### 5. 可视化与监控
- **数据仪表盘**：实时展示库存状态、预测结果、补货计划等关键指标
- **可视化分析**：提供多种图表和报表，支持数据深度分析
- **异常监控**：自动识别预测异常和库存异常，及时告警

### 核心技术组件

1. **数据处理引擎**：基于Pandas和Spark的大规模数据处理能力
2. **预测模型框架**：集成Scikit-learn、StatsModels、Prophet等机器学习库
3. **优化引擎**：基于PuLP的混合整数线性规划求解器
4. **特征存储**：基于Redis和PostgreSQL的特征管理系统
5. **API服务**：基于FastAPI的高性能REST API
6. **缓存系统**：基于Redis的分布式缓存
7. **监控系统**：集成Prometheus和Grafana的监控解决方案
